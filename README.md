# Maqnin OS

Maqnin or carduelis, (chardonneret in french) is a little script made to update Parrot OS with latest packages and scripts

## Requirements

Parrot OS 6.2

## Installation

clone the repo

chmod+x install.sh

You're good to go

## List of softwares and packages

- Update of subfinder
- Install of httpx
- Clone of XSSStrike
